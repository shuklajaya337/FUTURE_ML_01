# Sales & Demand Forecasting (Machine Learning Task 1)

##  Project Overview
This project predicts future sales based on historical data using Machine Learning (Random Forest Regressor). It was built for **Future Interns Machine Learning Task 1 (2026)**.

##  Features
- **Synthetic Data Generation**: Creates realistic sales data with trends and seasonality.
- **Data Cleaning**: Handles missing values automatically.
- **Feature Engineering**: Adds time-based features (Lag, Rolling Mean, etc.).
- **Modeling**: Uses Random Forest for robust forecasting.
- **Visualization**: Generates `sales_forecast_plot.png` and `feature_importance.png`.
- **Business Insights**: Prints a clear summary for stakeholders.
